# StringTemplateMapper

Class for map String value to string template.

Map object attribute to string templete
Example.

---------String Templete---------

Member id : ${idNumber} has been registerd.

to

---------String Message-----------

Member id : MB0020 has been registerd.



## Built With

* Spring Framework
* Java
* Intelij
